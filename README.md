# Shasavistic Chord Editor

This is a web-based editor for chord diagrams and symbols based on LΛMPLIGHT's Shasavistic notation system and theory (see [Caftaphata](https://youtu.be/cMnuMjXeHrY)).

Use the buttons in the right side panel to select a dimension and whether the interval should be ascending/descending. Then click in the workspace to add the interval to the chord.
